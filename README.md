Get [`jbang`](https://jbang.dev) (you can get it from `sdkman` for instance). Then you can run the Spring Boot CLI:

```
$ jbang spring@scratches version
Spring CLI v2.4.5
```

or a vanilla config server:

```
$ jbang configserver@scratches
```
